🧬 Genetics 30–40% of exam

DNA, RNA, protein synthesis, mutations, inheritance patterns, pedigrees, and biotechnology

🔬 Structure of DNA vs. RNA

DNA

  • Double-stranded helix
  • Sugar: deoxyribose
  • Bases: A, T, G, C (Adenine, Thymine, Guanine, Cytosine)
  • Found in the nucleus
  • Stores genetic information permanently
  • A pairs with T (2 H-bonds), G pairs with C (3 H-bonds)

RNA

  • Single-stranded
  • Sugar: ribose
  • Bases: A, U, G, C (Uracil replaces Thymine)
  • Found in nucleus AND cytoplasm
  • Carries instructions for protein building
  • Three types: mRNA, tRNA, rRNA
🔑 Chargaff's Rules: In any DNA molecule, %A = %T and %G = %C. This is the basis of complementary base pairing.

🔁 DNA Replication

Goal: Copy DNA before cell division so each daughter cell gets a complete set of chromosomes.

Key Steps (in order)

  1. Helicase unwinds and "unzips" the double helix by breaking hydrogen bonds between base pairs.
  2. Each strand serves as a template for a new complementary strand.
  3. A short RNA primer attaches to start the new strand (DNA polymerase can't start from scratch).
  4. DNA polymerase reads the template 3'→5' and builds the new strand 5'→3', adding complementary nucleotides.
  5. The leading strand is made continuously; the lagging strand is made in fragments (Okazaki fragments).
  6. DNA ligase joins the fragments together.
  7. Result: 2 identical DNA molecules, each with one old strand + one new strand = semiconservative replication.
⚠️ Why errors are rare: DNA polymerase has a "proofreading" function — it checks each new base and removes mistakes. Error rate: ~1 in 10 billion bases.
5' to 3' Direction

The new strand is ALWAYS built in the 5'→3' direction. The template is read 3'→5'. This is critical — the lagging strand must be built in fragments because it runs antiparallel.

Why Semiconservative?

Each new DNA molecule keeps one original (conserved) strand and one newly made strand. "Semi" = half conserved. Confirmed by Meselson-Stahl experiment.

🔗 DNA Replication & Cell Division Connection

  • DNA replication happens during S phase (Synthesis phase) of Interphase — BEFORE mitosis or meiosis begins.
  • Without replication, daughter cells would only get half the chromosomes.
  • Mitosis → body cells (somatic), produces 2 identical diploid cells (2n)
  • Meiosis → sex cells (gametes), produces 4 genetically unique haploid cells (n)
  • Meiosis is responsible for genetic variation through crossing over and independent assortment.

📜 Transcription: DNA → mRNA

Location: Nucleus · Template: DNA · Product: mRNA

Steps

  1. RNA polymerase binds to the promoter region on DNA.
  2. The DNA double helix unwinds locally.
  3. RNA polymerase reads the template strand 3'→5' and builds mRNA 5'→3' using RNA nucleotides (A, U, G, C).
  4. When it reaches the terminator sequence, it stops and releases the mRNA.
  5. The mRNA is processed: introns (non-coding) are spliced out, exons (coding) are joined.
  6. A 5' cap and poly-A tail are added for protection and export.
💡 Key rule: A→U, T→A, G→C, C→G when making mRNA from DNA template. Remember: Thymine in DNA becomes Uracil in RNA.

🏭 Translation: mRNA → Protein

Location: Ribosome (cytoplasm) · Template: mRNA · Product: Polypeptide/Protein

Key Players

mRNA

Carries the genetic code from nucleus to ribosome. Read in 3-base groups called codons.

tRNA

Brings amino acids to the ribosome. Has an anticodon that pairs with mRNA codon, and carries the correct amino acid on the other end.

rRNA

Makes up the ribosome structure. Has enzymatic activity (ribozyme) that forms peptide bonds between amino acids.

Steps of Translation

  1. Initiation: Ribosome assembles at the start codon (AUG), which codes for Methionine. Small and large ribosomal subunits join.
  2. Elongation: tRNA brings amino acids, anticodon matches mRNA codon, peptide bond forms between amino acids. Ribosome moves 3 bases down mRNA (5'→3').
  3. Termination: Ribosome reaches a stop codon (UAA, UAG, UGA). No tRNA matches these. Release factor protein frees the polypeptide chain.
  4. Polypeptide folds into a functional protein.
🛑 Stop codons: UAA, UAG, UGA → No amino acid; they signal "end of protein." Start codon: AUG → Always codes for Methionine (Met).

🔤 Reading the Genetic Code: Codons → Amino Acids

Each codon = 3 mRNA bases = 1 amino acid. There are 64 possible codons but only 20 amino acids → the code is redundant/degenerate (multiple codons can code for the same amino acid).

🔄 Central Dogma Summary

DNA → (transcription) → mRNA → (translation) → Protein

Remember the full chain: DNA → mRNA → tRNA (carrying amino acids) → Amino acid chain → Functional Protein

Enzymes involved: RNA polymerase (transcription), Ribosomes + rRNA (translation), tRNA (brings amino acids)

Point mutations affect a single nucleotide base in the DNA sequence.

🧩 Chromosomal Mutations

These affect large segments of chromosomes — more genes are affected than point mutations.

Deletion

A segment of chromosome is LOST. Fewer genes. Example: Cri du chat syndrome (deletion of chromosome 5p).

Duplication

A segment is COPIED twice on the same chromosome. Extra genes — can lead to overexpression.

Inversion

A segment is FLIPPED 180° and reinserted. Genes present but in wrong order.

Translocation

A segment MOVES to a different chromosome. Can disrupt gene function. Example: Philadelphia chromosome in leukemia.

Nondisjunction

Chromosomes fail to separate during meiosis → gamete has wrong number. Example: Trisomy 21 (Down Syndrome).

Comparing to Point Mutations

Chromosomal = large scale, many genes. Point = one base, possibly one amino acid. Chromosomal mutations are more likely to be lethal.

🌡️ Effects on Phenotype

Positive / Beneficial

Mutation improves survival or reproduction. Example: mutation giving antibiotic resistance to bacteria. Rare in complex organisms. These are selected FOR in evolution.

Neutral / Silent

No change in phenotype. Silent mutations (same amino acid) or mutations in non-coding regions. Neither selected for nor against.

Negative / Harmful

Reduces fitness. Most mutations fall here. Examples: sickle cell disease (missense), cancer-causing mutations (frameshifts in tumor suppressors). Selected AGAINST.

Law of Independent Assortment

Alleles for different genes sort independently into gametes (as long as the genes are on different chromosomes). This is why we multiply probabilities for two independent traits.

💡 Probability shortcut: P(round) from Rr×Rr = 3/4. P(yellow) from Yy×Yy = 3/4. P(round AND yellow) = 3/4 × 3/4 = 9/16

🔗 X-linked Inheritance

Some genes are located on the X chromosome. Because males (XY) only have one X, they express X-linked recessive traits more often.

Female genotype possibilities

  • XBXB = homozygous dominant (normal)
  • XBXb = heterozygous carrier (normal phenotype, carries recessive)
  • XbXb = homozygous recessive (affected)

Male genotype possibilities

  • XBY = normal
  • XbY = affected (only one X, so recessive is expressed)
🔑 X-linked recessive diseases: Colorblindness, Hemophilia A, Duchenne Muscular Dystrophy. More common in males. Fathers pass X-linked genes to daughters, not sons. Mothers pass X-linked genes to both sons and daughters.

🌳 Reading Pedigrees

A pedigree is a family tree showing which individuals are affected by a trait across generations.

Symbols

  • ⬜ Square = male, unaffected
  • ⬛ Filled square = male, affected
  • ⭕ Circle = female, unaffected
  • ⬤ Filled circle = female, affected
  • ⬤ with dot inside = carrier female
  • Horizontal line = mating pair
  • Vertical/diagonal line = offspring
  • Roman numerals = generations (I, II, III)

How to determine inheritance pattern

  1. Is it autosomal or X-linked? (Check if males or females are more affected)
  2. Is it dominant or recessive? (Does it skip generations? Are unaffected carriers possible?)
  3. If two unaffected parents have affected child → autosomal recessive
  4. If affected father has all affected daughters → X-linked dominant
🔎 In the pedigree above: Two unaffected parents (I) have an affected child (II-1) → must be autosomal recessive. Carrier mother (dot) passes the recessive allele. Grandparent (I-2 female) was affected, meaning she was homozygous recessive.

🧫 Gel Electrophoresis

What it does

Separates DNA fragments (or proteins) by size using an electrical current through a gel matrix. Used for DNA fingerprinting, paternity testing, forensics, and identifying genetic mutations.

How to read a gel

  • DNA is negatively charged → migrates toward positive electrode (bottom)
  • Smaller fragments travel farther (less resistance through gel)
  • Larger fragments stay near the top (more resistance)
  • Ladder (standard) = known size reference fragments
  • Bands at same position = same size fragment
  • Matching banding pattern = genetic match

Gel diagram (simplified)

Sample A and C match! Same banding pattern.

Applications: Paternity testing (child's bands come from parents), forensic crime scene identification, diagnosing genetic diseases, confirming cloned organisms, RFLP analysis for genetic variation.

🔬 Genetic Technologies Overview

You need to know the basics of what each technology does and its applications — not the detailed mechanism.

PCR – Polymerase Chain Reaction ▼
What: Amplifies (makes millions of copies of) a specific DNA segment. How: Heat to separate strands, cool to attach primers, DNA polymerase extends. Repeated cycles = exponential copying. Uses: Crime scene forensics (tiny DNA sample), disease diagnosis, COVID-19 testing, ancestry analysis. Why it matters: Can work with extremely tiny amounts of DNA.
Gel Electrophoresis ▼
What: Separates DNA (or proteins) by size using electrical current through gel. Uses: DNA fingerprinting, paternity testing, forensics, genetic disease diagnosis. Key: Smaller fragments move farther; compare to a ladder of known sizes.
Recombinant DNA / Genetic Engineering ▼
What: Combining DNA from two different organisms. How: Use restriction enzymes to cut DNA at specific sequences, insert into a vector (plasmid), introduce into host cell. Uses: Insulin production (bacteria make human insulin), crops with pest resistance, bacteria that make enzymes. Result: Host cell expresses the foreign gene.
GMOs – Genetically Modified Organisms ▼
What: Organisms with DNA from another species added to their genome. Examples: Bt corn (bacterial toxin gene → insect resistance), golden rice (Vitamin A gene), herbicide-resistant soybeans. Concerns: Unknown long-term effects, biodiversity, "superweeds." Benefits: Higher yields, disease resistance, nutritional improvement.
Cloning ▼
What: Creating a genetically identical copy of an organism. How (somatic cell nuclear transfer): Remove nucleus from egg cell, replace with nucleus from donor body cell, stimulate to divide, implant in surrogate. Examples: Dolly the sheep (first cloned mammal). Uses: Preserving endangered species, medical research. Ethical concerns: Reproductive vs. therapeutic cloning debate.
CRISPR-Cas9 (Gene Editing) ▼
What: Precise "molecular scissors" that can cut, edit, or delete specific genes. Guide RNA directs Cas9 protein to the target DNA sequence. Uses: Correcting genetic diseases (sickle cell trials), agriculture, research. Concerns: Germline editing ethics (heritable changes), off-target effects.
DNA Sequencing & Human Genome Project ▼
What: Determines the exact order of bases in a DNA molecule. Human Genome Project: Mapped all ~3 billion base pairs in human DNA (completed 2003). Applications: Understanding genetic diseases, personalized medicine, ancestry tracing, finding disease-causing genes.
Gene Therapy ▼
What: Inserting a functional gene into a patient's cells to treat a genetic disease. Example: Treating severe combined immunodeficiency (SCID). Challenge: Getting the gene into the right cells without immune rejection. Viral vectors often used as delivery systems.

🌍 Darwin's Theory of Evolution by Natural Selection

Natural selection = the process by which organisms with favorable traits survive and reproduce more successfully than those without, leading to changes in population over time.

4 Key Observations / Postulates

1. Overproduction

Organisms produce more offspring than can possibly survive. Competition for resources is inevitable.

2. Variation

Individuals in a population show heritable variation in traits. Without variation, there's nothing to select upon.

3. Struggle for Existence

Limited resources mean not all offspring survive. Competition happens within and between species.

4. Survival of the Fittest

Individuals with traits better suited to their environment survive and reproduce more → "fitness" = reproductive success, not physical strength.

5. Inheritance

Favorable traits are passed to offspring. Over generations, favorable traits become more common in the population.

Result: Adaptation

Over many generations, populations become better adapted to their environment. Enough change = new species.

🦴 Evidence for Evolution

Fossil Evidence

  • Transitional fossils show intermediate forms between ancestral and descendant species
  • Fossils in rock layers show change over time (older = deeper layers)
  • Gaps in the fossil record are expected — fossilization is rare

Biogeography

  • Related species found in same geographic area (islands near continent)
  • Explains distribution of similar organisms on continents once connected

Anatomical Evidence

  • Homologous structures: Same structure, different function → common ancestor (e.g., human arm, whale flipper, bat wing — all are modified forelimbs)
  • Analogous structures: Different structure, same function → convergent evolution (e.g., bird wing vs. butterfly wing)
  • Vestigial structures: Remnant of an ancestral structure no longer used (e.g., human tailbone, whale pelvis bones)

Molecular Evidence

  • DNA and protein sequences compared across species
  • More similar sequence = more closely related
  • Cytochrome c protein nearly identical in related species
🔑 Key distinction: Homologous = same underlying structure (common ancestor). Analogous = same function but different structure (convergent evolution). Don't confuse these!

🌱 Artificial Selection vs. Natural Selection

Artificial Selection (Selective Breeding)

Humans choose which individuals reproduce based on desired traits. Examples: dog breeds, crop varieties, domesticated animals. Faster than natural selection but same mechanism — selecting favorable traits.

Natural Selection

Environment (not humans) determines which individuals survive and reproduce. Acts on existing variation. Slower but produces adaptation to actual environmental conditions.

📊 Three Patterns of Natural Selection

These describe which phenotypes in a population are selected for or against. Think about what happens to the bell curve of trait distribution.

Stabilizing Selection

Extreme phenotypes are selected AGAINST. Middle (average) phenotype is favored. Result: LESS variation, population stays the same. Example: Human birth weight — very heavy and very light babies have lower survival rates.

Directional Selection

One extreme phenotype is favored over the other. Result: population SHIFTS in one direction. Example: Antibiotic resistance — bacteria resistant to antibiotics survive and reproduce; population shifts toward resistance. Peppered moth during industrial revolution.

Disruptive Selection

Both extreme phenotypes are favored; middle is selected against. Result: MORE variation, population may split into two groups. Example: Darwin's finches with large or small beaks outcompete medium-beak birds when seeds are either very large or very small.

📈 Tip for the exam: If asked which type of selection reduces variation → Stabilizing. Shifts the average → Directional. Creates two peaks/groups → Disruptive.

💪 Biological Fitness

  • Fitness = an organism's ability to survive AND reproduce in its environment
  • NOT about physical strength — a weak organism that reproduces 10 times is more "fit" than a strong one that never reproduces
  • Fitness is always relative to the environment — what's fit in one environment may not be in another
  • Natural selection acts on phenotypes but selects the underlying genotypes
  • Fitness increases as beneficial traits increase in frequency across generations

🌐 Speciation

A species = a group of organisms that can interbreed and produce fertile offspring. Speciation = the process by which new species form, usually through reproductive isolation.
Biological Species Concept

Species defined by ability to interbreed and produce fertile offspring in nature. Doesn't work for asexual organisms or fossils.

Evolutionary Species Concept

Species defined by shared evolutionary lineage and unique ecological roles. Applies to fossils and asexual organisms too.

↔️ Convergent vs. Divergent Evolution

Convergent Evolution

Unrelated species develop similar traits due to similar environmental pressures. Produces analogous structures. Example: Dolphins and sharks both have streamlined bodies and fins — but dolphins are mammals, sharks are fish. Same shape, different ancestry.

Divergent Evolution

Related species evolve different traits as they adapt to different environments. Produces homologous structures. Example: Darwin's finches — one ancestor, many beak shapes based on food source. All are closely related but look/behave differently.

🧬 Population Genetics: Forces of Evolution

Gene Flow
Movement of alleles between populations due to migration. Tends to make populations MORE SIMILAR. Reduces genetic differences between populations.
Genetic Drift
Random changes in allele frequency due to chance, especially in SMALL populations. NOT natural selection — random, not adaptive. Can eliminate alleles by chance.
Bottleneck Effect
Dramatic reduction in population size (disaster/hunting) → surviving population has LOW genetic diversity. Example: Cheetahs — genetically nearly identical due to ancient bottleneck.
Founder Effect
A small group leaves to start a new population. New population has LESS genetic diversity. Example: Amish community with high rate of rare genetic disorders due to small founding group.
Genetic Equilibrium (Hardy-Weinberg)
A population where allele frequencies DON'T change over time. Requires: no mutation, no migration, large population, random mating, no natural selection. Rarely occurs in nature — evolution is happening.
Mutation
Source of ALL new genetic variation. Without mutation, no new alleles. Provides the raw material for natural selection and evolution.
⚠️ Genetic drift vs. Natural selection: Drift = random (small populations). Selection = non-random, adaptive (based on fitness). Both change allele frequencies but for different reasons.

🌳 Reading Cladograms & Phylogenetic Trees

A cladogram shows evolutionary relationships based on shared derived characteristics. A phylogenetic tree adds time (branch length can = time or genetic change).

How to read a cladogram

  1. Species on the tips (branches) are the living/modern species being compared.
  2. Nodes (where branches split) represent common ancestors.
  3. Two species that share a more recent common ancestor are more closely related.
  4. Derived characters (new traits) mark what separates groups — they're listed along the branches where they first appeared.
  5. To find which two species are most closely related: find the node they share that is closest to the tips.
📌 Exam tip: "Which two organisms are most closely related?" = Find the pair that shares the most recent (lowest on the tree / rightmost) common ancestor node.

⏱️ Molecular Clock

  • The idea that mutations accumulate in DNA at a roughly constant rate over time.
  • More DNA differences = more time since common ancestor = less related.
  • Used to estimate when species diverged (split from a common ancestor).
  • Example: Comparing cytochrome c protein sequences across species to build evolutionary trees.

🦴 Human Evolution — Key Adaptations

The exam may ask: What evolutionary changes allowed humans to become the dominant species? Focus on: bipedalism, brain size, tool use, and language.

Bipedalism (Walking Upright)

  • Freeing of hands for tool use and carrying
  • Evidence in skeleton: S-curved spine, wider pelvis, angled femur (thigh bone angles inward), feet with arch and non-opposable big toe
  • Foramen magnum (hole at base of skull) directly underneath skull in bipeds — on the back in quadrupeds
  • Knee joint angle differs from quadruped knee

Brain Development

  • Skull volume (cranial capacity) increased dramatically over hominid evolution
  • Larger, more complex brain → higher cognitive function, language, problem solving
  • Smaller brow ridges, larger braincase, flatter face over time in fossil record

Tool Use & Cultural Evolution

  • Tool use accelerated human success beyond biological evolution
  • Allowed humans to hunt larger animals, process food, build shelter
  • Language and communication allowed cultural transmission of knowledge
  • Fire use enabled cooking → more calorie-dense food → supported large brain

Global Migration

  • Originated in Africa (fossil evidence, DNA evidence)
  • Migrated out of Africa to populate all continents
  • Reached all habitable continents by ~15,000 years ago
  • Language, social cooperation, tools enabled survival in diverse environments

🔭 Levels of Ecological Organization

Organism
A single individual (one deer, one oak tree)
Population
All members of the SAME species in an area (all deer in a forest)
Community
All different species in an area interacting (deer + wolves + trees + bacteria)
Ecosystem
Community PLUS the abiotic (nonliving) environment (soil, water, sunlight, temperature)
Biome
Large geographic area with characteristic climate and organisms (tropical rainforest, tundra, desert)
Biosphere
All living things and their environments on Earth — the global ecosystem

🤝 Types of Symbiosis

Mutualism (+/+)

BOTH species benefit. Example: Bees and flowers (bee gets nectar, flower gets pollinated). Clownfish and sea anemone. Nitrogen-fixing bacteria in legume roots.

Commensalism (+/0)

One species benefits, the other is UNAFFECTED. Example: Barnacles on a whale (barnacles get transport, whale is unaffected). Remora fish on sharks.

Parasitism (+/−)

One species benefits, the other is HARMED (but usually not killed quickly). Example: Tapeworms in host intestines. Ticks on deer. Mistletoe on trees.

🌱 Food Chains, Food Webs & Trophic Levels

A food chain shows one path of energy flow. A food web shows ALL possible feeding relationships in an ecosystem — more realistic, more complex.

10% Rule of Energy Transfer

Only about 10% of energy is passed from one trophic level to the next. The other 90% is lost as heat (respiration), waste, or not consumed. This is why food chains rarely have more than 4–5 levels — not enough energy remains.

Ecological Pyramids

  • Energy pyramid: Always wide at bottom (producers), narrows going up. Shows 10% energy transfer. Can NEVER be inverted.
  • Biomass pyramid: Usually wide at bottom. Shows mass of organisms at each level. Can occasionally be inverted (e.g., phytoplankton vs. zooplankton).
  • Numbers pyramid: Can be inverted (one oak tree supports thousands of insects).

🔑 Keystone Species & Trophic Cascade

Keystone Species

A species that has a disproportionately large effect on its ecosystem relative to its abundance. Remove it → ecosystem dramatically changes or collapses. Example: Sea otters control sea urchin populations → protect kelp forests. Wolves in Yellowstone (trophic cascade).

Trophic Cascade

When removing one species causes a chain reaction through the food web. Example: Remove wolves → deer overpopulate → overgrazing → vegetation destroyed → soil erosion → rivers change. Adding wolves back reverses all these effects.

📈 Population Growth Patterns

Exponential Growth (J-curve)

  • Population grows at an ever-increasing rate
  • Occurs when resources are unlimited
  • Produces a J-shaped curve
  • Population size is not limited by carrying capacity
  • Common in: bacteria, organisms in new environments, invasive species

Logistic Growth (S-curve / sigmoidal)

  • Growth slows as population approaches carrying capacity (K)
  • Produces an S-shaped curve
  • Growth is fast when population is small, slows as resources become limiting
  • More realistic model for most populations
📌 Carrying capacity (K) = the maximum population size an environment can sustainably support. Determined by food, water, space, and other resources. Logistic growth levels off AT carrying capacity.

⚖️ Limiting Factors

Density-Dependent Factors

Effect increases as population DENSITY increases.

  • Food competition
  • Disease (spreads faster in crowded populations)
  • Predation (more prey = more predators)
  • Parasitism
  • Intraspecific competition

Density-Independent Factors

Effect is the same regardless of population density.

  • Natural disasters (hurricanes, floods)
  • Extreme temperature/weather
  • Drought or flooding
  • Human activities (pollution, habitat destruction)

📊 Survivorship Curves

Type I (Convex)

Most individuals survive to old age, then die rapidly. Examples: Humans, elephants, large mammals. Few offspring, high parental care. K-selected species.

Type II (Diagonal/Linear)

Constant mortality rate throughout life — equal chance of dying at any age. Examples: Birds, squirrels, some reptiles. Moderate offspring number.

Type III (Concave)

Most individuals die young, but survivors live long. Examples: Fish, oysters, sea turtles, plants. Many offspring, little parental care. R-selected species.

🔑 r vs K species: r-selected = many offspring, little care, short life, exponential growth (Type III curve). K-selected = few offspring, high care, long life, logistic growth (Type I curve).

👥 Age Structure Diagrams (Population Pyramids)

Wide Base (Expanding – LDC)

Many young people, fewer old. Found in Less Developed Countries (LDCs). Indicates RAPID POPULATION GROWTH. High birth rate, high death rate among elderly. Example: Many African and Asian countries.

Narrow Base (Stable/Shrinking – MDC)

More uniform age distribution, or narrower base. Found in More Developed Countries (MDCs). Indicates stable or declining population. Low birth rate, lower death rate. Example: Germany, Japan, Italy.

☠️ Biomagnification & Bioaccumulation

These are HIGH-PRIORITY exam topics — know the difference!
Bioaccumulation

The buildup of a toxic substance in a SINGLE ORGANISM's tissues over time (usually fat-soluble toxins that aren't easily broken down). Example: A fish absorbs mercury from water over its lifetime → mercury builds up in its tissues.

Biomagnification

The INCREASING concentration of toxins at each trophic level as you move UP the food chain. Because organisms at higher levels eat many organisms below them, they concentrate the toxin further. Top predators are most affected.

📌 Classic example: DDT & Bald Eagles
Plankton absorb small amounts of DDT from water → Small fish eat many plankton → accumulate more DDT → Large fish eat many small fish → Bald eagle eats many large fish → MASSIVE DDT concentration → Thin eggshells → Near extinction of eagles.

Concentration increase: Water: 0.000003 ppm → Plankton: 0.04 ppm → Small fish: 0.5 ppm → Large fish: 2 ppm → Eagles: 25 ppm (over 8 million times more concentrated!)
⚠️ Toxins that biomagnify: PCBs, DDT, mercury (methylmercury), lead, dioxins. These are fat-soluble, persistent (don't break down), and accumulate in fatty tissues.

🌡️ Greenhouse Effect & Global Climate Change

Natural Greenhouse Effect (Good!)

  • Greenhouse gases (CO₂, H₂O, CH₄, N₂O) trap heat in the atmosphere
  • Without it, Earth's average temp would be -18°C (0°F) — too cold for life
  • Sunlight enters → absorbed by Earth → radiated as heat → GHGs trap some heat → Earth stays warm

Enhanced Greenhouse Effect (Problem!)

  • Human activities add excess GHGs → trap more heat than normal → global warming
  • CO₂: Burning fossil fuels (cars, factories, power plants), deforestation
  • Methane (CH₄): Livestock, landfills, rice paddies, natural gas
  • Nitrous oxide: Fertilizers, agriculture
  • Chlorofluorocarbons (CFCs): Old refrigerants, aerosols → also destroy ozone

Effects of Global Climate Change

  • Rising global average temperatures
  • Melting polar ice and glaciers → rising sea levels
  • More extreme weather events (hurricanes, droughts, floods)
  • Ocean acidification (CO₂ dissolves in water → carbonic acid)
  • Disruption of seasonal patterns → affects migration and reproduction
  • Habitat loss → biodiversity loss → species extinction
  • Coral bleaching due to warming oceans

🌳 Deforestation & Its Effects

Carbon Cycle Disruption

Trees store carbon in biomass. Cutting trees releases CO₂ back to atmosphere. Fewer trees = less CO₂ absorbed. Double impact: releases stored carbon AND removes carbon sink.

Water Cycle Disruption

Trees release water vapor through transpiration. Deforestation reduces local rainfall, disrupts water cycles, increases flooding and erosion (no roots to hold soil), reduces groundwater recharge.

Biodiversity Loss

Tropical rainforests hold ~50% of Earth's species. Destroying habitat → extinction of species, many before discovered. Loss of potential medicines, food sources, and ecosystem services.

Soil Degradation

Rainforest soil is thin and nutrient-poor. Most nutrients are in the living biomass. Without trees, soil erodes rapidly → desertification → land can no longer support agriculture.

Climate Feedback Loop

Deforestation → more CO₂ → warmer temps → more forest fires → more CO₂ → positive feedback loop that accelerates climate change.

Indigenous Impact

Destroys habitats of indigenous peoples, disrupts traditional food systems, affects communities dependent on forest resources.

🎯 Overexploitation & Tragedy of the Commons

Tragedy of the Commons: When a shared resource is available to all, each individual is incentivized to use as much as possible for personal gain, even though overuse destroys the resource for everyone. The "rational" individual choice leads to collective ruin.

Examples of Overexploitation

  • Overfishing → collapse of fish populations (Atlantic cod, bluefin tuna)
  • Hunting large mammals to near-extinction (rhinos, elephants for ivory)
  • Overharvesting of timber, wild plants
  • Overgrazing by livestock → desertification

Solutions / Conservation Approaches

  • Fishing quotas and catch limits
  • Marine protected areas and wildlife refuges
  • International agreements (CITES for endangered species trade)
  • Community-based management of shared resources
  • Consumer choices (sustainable seafood certification)

🌍 Why Conservation Matters

Direct values of biodiversity

  • Medical: 25% of pharmaceuticals come from plants (aspirin from willow, penicillin from fungus)
  • Agricultural: Wild plants provide genetic diversity for crop improvement
  • Economic: Ecotourism, fishing, timber, agriculture
  • Scientific: Understanding ecosystems helps us manage them

Ecosystem Services (Indirect)

  • Air and water purification
  • Carbon sequestration (storing CO₂)
  • Pollination of crops (bees, butterflies)
  • Nutrient cycling
  • Flood control by wetlands
  • Soil stabilization by plants
💡 Biodiversity has three levels: Species diversity (number of different species), Genetic diversity (variety within species), Ecosystem diversity (variety of habitats). Losing any level weakens overall resilience.

🔬 Other Human Impacts

Ozone Depletion

  • Ozone (O₃) in stratosphere blocks harmful UV radiation
  • CFCs (chlorofluorocarbons) break down ozone molecules
  • Ozone hole over Antarctica detected in 1980s
  • More UV → increased skin cancer, cataracts, damage to marine phytoplankton
  • Montreal Protocol (1987) banned CFCs → ozone slowly recovering

Acid Rain / Acid Deposition

  • Burning fossil fuels releases SO₂ and NO₂
  • Combine with water vapor → sulfuric and nitric acids
  • Falls as acid rain (pH 4–5 instead of normal 5.6)
  • Effects: kills fish and aquatic life, damages forests, corrodes buildings
  • Crosses national borders (e.g., US emissions affect Canadian lakes)
Eutrophication: Excess nutrients (nitrogen, phosphorus from fertilizer runoff) enter water → algae bloom → algae die → bacteria decompose them → bacteria use up all oxygen → dead zone (hypoxic zone). No oxygen = no fish. Example: Gulf of Mexico dead zone from Mississippi River agricultural runoff.

📊 Interpreting Ecology Graphs — Exam Tips

Predator-Prey Graphs

  • Prey population peaks BEFORE predator population (predators lag behind prey)
  • When prey increases → more food for predators → predators increase
  • When predators increase → more prey eaten → prey decreases
  • When prey decreases → predators starve → predators decrease
  • Classic oscillating (wave) pattern on graph

Population Growth

  • J-curve = exponential growth (unlimited resources)
  • S-curve = logistic growth (resource limits)
  • Leveling off = at or near carrying capacity
  • Sudden drop = limiting factor applied (disease, predator, disaster)

Energy/Biomass Pyramids

  • Wide base = producers (most energy/biomass)
  • Each level = 10% of energy from level below
  • If 10,000 kcal in producers → 1,000 in herbivores → 100 in carnivores → 10 in top predator
  • WHY less energy at each level: heat loss from respiration, waste, incomplete consumption

Biomagnification Graph

  • Toxin concentration INCREASES going up food chain
  • Even if water has tiny amount, top predator accumulates huge dose
  • Graph usually shows exponential increase up trophic levels
📌 Key graph interpretation skills:
  • Identify independent variable (x-axis) and dependent variable (y-axis)
  • Identify trends (increasing, decreasing, cyclical, leveling off)
  • Identify the cause of any sudden changes (usually a specific ecological event)
  • Compare multiple lines on the same graph (predator vs. prey, two species)
  • Calculate percentage change or ratios when asked
